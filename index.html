How to load real businesses

This app ships with generated sample data so every feature works immediately. To search real businesses, you have two real options:

  1. Live Google Search (built in): connect your own Google Maps API key in the sidebar, with the Places API enabled and billing set up in Google Cloud Console. Then pick a country/city and business type and click "Search Google now."
  2. CSV import: export business data from a legitimate source (the Places API directly, or a licensed data provider) into a spreadsheet matching the template, then use "Import CSV."

Direct scraping of Google Maps is against Google's terms of service and isn't something this app does. The Places API is the legitimate way to get this same data — it even tells you directly whether a business has a website on file.

Getting a Google Maps API key

  1. Go to the Google Cloud Console and create a project (or use an existing one).
  2. Enable billing on the project — Google requires this for the Places API, though it includes a recurring free usage credit.
  3. Under "APIs & Services," enable the Maps JavaScript API and the Places API.
  4. Create an API key under "Credentials," and (recommended) restrict it to those two APIs and to your own domain.
  5. Paste that key into the box in the sidebar and click Connect.

You're billed directly by Google for what you use — this app doesn't add any markup or proxy your requests.
