How to load real businesses

This app ships with generated sample data so every feature works immediately. To search real businesses:

  1. Use the Google Places API (Text Search or Nearby Search, then Place Details) for the countries/cities and categories you care about. The Place Details response includes a website field — if it's missing, that's a business without a website.
  2. Export those results into a spreadsheet matching the column layout in the CSV template (use the Download CSV template button in the toolbar).
  3. Use Import CSV to load that file in. Every filter, search, and pagination feature will then run on your real data.

This keeps the data source fully compliant with Google's terms — direct scraping of Google Maps is against their policies and isn't something this app does.
